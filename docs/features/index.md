# 功能特性

Oh My OpenCode Agents 提供了丰富的功能，帮助您快速搭建和管理 Ubuntu 服务器开发环境。

## 本章节内容

- **[功能概览](./overview.md)** - 所有功能的总体介绍
- **[开发工具](./dev-tools.md)** - Git、Docker、Neovim 等工具配置
- **[服务器管理](./server-management.md)** - 服务器监控和管理功能
- **[Nginx集成](./nginx-integration.md)** - Web 服务器配置指南

## 主要特性

!!! tip "核心优势"
    一键安装、智能配置、生产就绪、易于维护。

### 自动化安装

- 交互式安装向导
- 自动模式支持（适合 CI/CD）
- 模块化组件选择
- 智能依赖检测

### 开发工具链

- **版本控制**: Git + GitHub CLI
- **容器化**: Docker + Docker Compose
- **编辑器**: Neovim 配置
- **运行时**: Node.js (nvm) + Python (Poetry)

### 服务器套件

- **管理面板**: 宝塔面板 / OpenCode Manager
- **Web服务器**: Nginx 反向代理
- **监控工具**: 系统资源监控
