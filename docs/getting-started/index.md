# 快速开始

欢迎来到 Oh My OpenCode Agents 的快速开始指南！

本部分将帮助您快速上手，在几分钟内完成项目部署。

## 本章节内容

- **[安装指南](./installation.md)** - 在不同环境下安装项目
- **[快速开始](./quick-start.md)** - 快速运行您的第一个实例
- **[首次运行](./first-run.md)** - 初始配置和常见问题

## 需要准备的

在开始之前，请确保您已经准备好：

- 一台运行 Ubuntu 的服务器（推荐 20.04 LTS 或 22.04 LTS）
- root 或具有 sudo 权限的用户
- 基本的命令行操作知识

## 快速命令

如果您已经准备好了环境，可以直接运行：

```bash
curl -fsSL https://raw.githubusercontent.com/choovin/oh-my-opencode-agents/main/install.sh | bash
```

这将自动安装所有必要的组件。
